¿Cómo Funciona CacaoScan?

CacaoScan combina visión computacional, análisis colorimétrico avanzado y datos de peso integrados mediante IoT para detectar tempranamente enfermedades como la moniliasis.

photo_camera

Captura de Imagen

El agricultor captura la mazorca con la cámara del teléfono. La aplicación normaliza condiciones de luz para un análisis estable.

palette

Colorimetría CIELAB

Se transforma la imagen al espacio CIELAB para identificar variaciones en tono y luminancia asociadas a madurez o infección.

texture

Análisis de Textura

Se evalúan patrones superficiales y brillo para detectar alteraciones que el ojo humano no percibe fácilmente.

scale

Integración con Peso

La balanza electrónica transmite el peso por Bluetooth. Este dato complementa el análisis visual.

psychology

Clasificación IA

El modelo entrenado con datos locales clasifica la mazorca con alta precisión.

verified

Reporte Inmediato

El sistema entrega diagnóstico instantáneo: Sana, Posible Infección o No Apta para comercialización.
